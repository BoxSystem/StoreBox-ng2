## 构建及使用

### NPM Run-Script

```bash
npm run build
```

### Docker Image

```bash
docker build -t boxsystem/storebox-ng2 .
```

或使用已构建好的镜像
```bash
docker pull yanqing07/storebox-ng2
```

### Docker-compose

```bash
docker-compose up --build -d
```
